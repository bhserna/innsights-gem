Overview

Generates the configuration files for innsights: * config/innsights.yml holds the app credentials and general configuration * config/initializers/innsights.rb (holds all the watched models and controllers)

def generate_config
  @config = Innsights::Client.create(processed_app_name)
  if @config.nil?
    abort
  else
    template 'innsights.yml', 'config/innsights.yml'
  end
end
