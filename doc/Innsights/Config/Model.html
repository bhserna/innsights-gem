Overview

Uses a DSL to configure a report in a Model and prepares it to run on a given callback the default callback is after create but it can be any kind of active record callback

def commit
  if valid?
		callback_process = setup(klass)
		klass.send(event, callback_process)
		klass
  else
		Innsights::ErrorMessage.log("#{klass} doesn't respond to callback event #{event}")
  end
end
